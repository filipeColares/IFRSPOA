1. O que é, como surgiu e o que faz o W3C 
Fundada por Tim Berners-lee em 1994, também criador da Word Wide Web, o W3C (Word Wide Web Consortium) é uma organização que tem a missão de desenvolver padrões de alta qualidade para a Word Wide Web.
Esses padrões visam melhorar a acessibilidade, interoperabilidade e usabilidade para os clientes/usuarios.

2. O que são "Padrões Web" (web standards)
Padrões web são documentos que definem e regulamentam as tecnologias em uso na web. Estes descrevem qual a melhor forma de utilização dessas tecnologias para garantir qualidade para o cliente final.

3. Descreva e exemplifique "Usabilidade" (a, b e c)
A) usabilidade é um termo utilizado para indicar a forma como o usuário utiliza as aplicações de forma clara e objetiva. Uma interface de boa usabilidade significa que o usuário não precisa de um manual para utilizá-la. 
B) A usabilidade, ou tornar uma interface usual, serve para facilitar o aprendizado, navegação, potencializar a satisfação para o usuário final.
C) Dicas importantes para se usar em uma interface usual são:
Utilizar a linguagem do usuário final
A interface deve permitir que o usuário faça suas escolhas, não que fique preso a regras de utilização da interface
A interface deve dar retornos para o usuário, a ponto de explicar o que está acontecendo em uma trasação de dados ou carregamento de informações mais pesado.
Prevenir erros de usabilidade e/ou o tratamento dos mesmos para a linguagem que o usuário usa.


4. Descreva e exemplifique "Acessibilidade" (a, b e c).
A) Acessibilidade é um termo utilizado para que interefaces como web sites estejam disponíveis para acesso de quaisquer usuários, idependete das suas limitações fisicas, intelectuais e classes sociais. 
B) Um site acessível permite que todas e qualquer pessoa possam perceber, entender, navegar, interagir e contribuir para a web
C) Dicas importantes para um site se tornar acessível:
Não bloqueie o zoom na página, pois pode ser que usuários com dificuldade em leitura necessitem para utilizar
Use a marcação HTML correta
Siga os padrões de acessibilidade da web

5. O que significa validação de página / código visando padrões Web.
Validação de página web/ código é uma análise feita no arquivo html da página para conferência se o mesmo está de acordo com o padrão correto orientado pelo W3C, orgão que regulamenta a internet.
